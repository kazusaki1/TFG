1) Abrir la consola de WinPython (WinPython Command Prompt)
2) Dirigirse a la ruta donde se encuentra restBackend. Por ejemplo: 
	cd C:\Users\Marc\Desktop\TFG\IonicAppWithRestBackend\backend\python_django\restBackend
3) Ejecutar el server mediante el comando: 
	python manage.py runserver 192.168.1.33:8000 (cambiar la IP por la tuya)
